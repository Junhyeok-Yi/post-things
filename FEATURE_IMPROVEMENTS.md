## 🎯 주요 기능 상세

### 회의록 모드 사용법

1. **회의록 모드 활성화**
   - 메모 입력 화면 상단의 "회의록 모드" 체크박스를 체크
   - 체크 후 작성하는 모든 메모가 회의록 카테고리로 저장됨

2. **회의록 작성**
   - 회의록 모드가 켜져 있는 동안 연속으로 메모 작성
   - 모든 메모가 같은 회의 ID로 그룹화됨

3. **회의록 모드 종료**
   - 체크박스를 해제하면 회의록 모드 종료
   - 다음에 다시 켜면 새로운 회의로 시작

4. **회의록 보기**
   - Affinity Diagram에서 "회의록" 탭 선택
   - 날짜, 시간, AI가 추출한 회의 제목과 함께 표시

### AI 태그 수정 기능

1. **AI 태그 확인**
   - 메모 작성 중 포스트잇 아래에 AI가 예측한 카테고리 표시
   - 예: "AI: To-Do"

2. **태그 수정**
   - "수정" 버튼 클릭
   - 원하는 카테고리 선택
   - 수정된 정보는 파인튜닝 데이터로 저장되어 향후 AI 정확도 향상에 활용

## 🔧 기술적 세부사항

### 회의록 그룹화 알고리즘
- 같은 `meetingId`를 가진 메모들을 그룹화
- 회의록 모드가 켜질 때 새로운 UUID 생성
- 회의록 모드가 꺼질 때까지 같은 ID 사용

### 회의 제목 추출 로직
- 메모 내용에서 공통 키워드 추출
- 빈도가 높은 키워드를 조합하여 제목 생성
- 메모가 1개인 경우 해당 메모 내용을 제목으로 사용

### 파인튜닝 데이터 구조
```typescript
interface FinetuningData {
  originalContent: string;
  aiPredictedCategory: string;
  userCorrectedCategory: string;
  timestamp: string;
}
```
